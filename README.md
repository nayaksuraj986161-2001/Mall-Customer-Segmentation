# Mall-Customer-Segmentation
Clean and prepare a raw dataset (with nulls, duplicates, inconsistent formats).
# Data Cleaning and Preprocessing Summary

The Mall Customer Segmentation dataset was cleaned and preprocessed using Python Pandas. The dataset was examined for missing values, and no missing values were found. Duplicate records were checked and no duplicate rows were detected. Column names were reviewed and standardized for consistency and ease of use. The data types of all columns were verified and found to be appropriate for analysis. The Gender column was inspected for formatting inconsistencies and was already standardized as "Male" and "Female". Finally, the cleaned dataset was exported as cleaned_mall_customers.csv.

## Key Changes

* Verified that no missing values were present
* Checked for duplicate records
* Verified that no duplicate rows existed
* Standardized column names
* Validated data types
* Inspected categorical values for consistency
